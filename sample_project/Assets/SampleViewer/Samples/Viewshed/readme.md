# Explore an interactive viewshed visualization

## How to use the sample

1. Open the scene.
2. Click on the **ArcGISMap** GameObject and set your API key in the **Inspector** window.
3. Click play.

## How it works

This interactive viewshed sample visualizes the areas of a terrain that are visible from a specific viewpoint.

The core of the sample is the `ViewshedObserver` script, which sets up an observer camera and creates a depth texture to capture the terrain's depth information. This script also updates global shader properties with the camera's view-projection matrix and far clip plane, which are essential for the custom shader to determine visibility.

The `ViewshedUIControl` script provides a user interface with sliders that allow you to adjust the observer camera's field of view, distance, rotation, and the opacity of the viewshed visualization. As you interact with these sliders, the script updates the observer camera and shader properties in real-time.

To enhance the user experience, the `SliderText` script updates text fields to display the current values of the sliders, providing immediate feedback. Additionally, the `UICameraDisabler` script ensures that the camera controller is disabled when the pointer is over a UI element, preventing unwanted camera movement during UI interactions.

The `Blit` script implements a custom render pass that applies a material (shader) to the camera's render target. This material uses a custom shader to highlight visible areas of the terrain based on the depth texture and other properties set by the `ViewshedObserver`.

The custom shader is designed to color the terrain based on visibility, with visible areas highlighted in green and invisible areas in red. By using the depth texture generated by the observer camera, the shader can accurately determine which areas are visible.

Together, these components create an interactive and dynamic viewshed visualization, allowing you to explore and understand the visibility of different areas of the terrain from a specific viewpoint.
